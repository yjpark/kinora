# Task: `kinora repack` — commit + clone-to-temp + atomic swap

**Goal.** Build `kinora::repack::repack_repo` that runs commit, clones into a temp
sibling, and atomically swaps it into place with rollback, plus a `kinora repack`
CLI.

**Realizes:**
- [[kinora-0caz:spec]] — `kinora repack` composes commit + clone-to-temp + atomic swap.
- [[kinora-b1mg:spec]] — `kinora clone` rebuilds `.kinora/` into a fresh directory.

**Plan:** [[kinora-0caz:plan]]

**Done when:** `kinora repack` on a repo with legacy files produces one with none,
render/resolve output is identical, no lingering temp dirs after success, a clear
error if a previous repack left state behind; zero warnings.

## Outcome
Landed in 3 commits. `repack_repo` does preflight (refuse on lingering
`.repack-tmp`/`.repack-old`), `commit_all` (bail if any root failed), clone-to-temp
sibling, two-rename swap with rollback on rename-2 failure, then unlinks the old
dir. CLI wrapper uses the normal `find_repo_root` walk-up; `-C` resolves in main
before dispatch. 6 library + 7 CLI tests. Deviations: git-dirty rail deferred,
rollback path not test-covered (both noted in spec).
