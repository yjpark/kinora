# Task: Wire fastrace spans + logforth; surface silent error sites

**Goal.** Wire fastrace + logforth into kinora-cli so library calls emit structured
spans and swallowed errors surface as debug logs, without changing the error enums
(rootcause migration is a separate follow-up).

**Done when:** `RUST_LOG=debug` surfaces `ExternalRefs::collect` skip logs,
`KINORA_TRACE=1` produces fastrace console output, all existing tests pass, zero
warnings.

## Outcome
- Deps: `kinora` gets `log` + `fastrace` (no `enable`, zero-cost without a CLI init);
  `kinora-cli` gets `log`, `fastrace` (with `enable`), `logforth`,
  `logforth-append-fastrace`. Cargo feature unification propagates `enable` so
  library `#[fastrace::trace]` attributes emit spans under the CLI.
- Instrumented entry points: `compact::{compact_root, compact_all,
  compact_root_with_refs, ExternalRefs::collect}`, `kino::store_kino`,
  `resolve::Resolver::load`, `init::init`, `ledger::Ledger::read_all_events`.
- Swallow sites: 5 `Err(_) => continue` in `ExternalRefs::collect` now
  `log::debug!` with source root + error context; two `parse_ts` fallbacks in
  `apply_root_entry_gc`/`prune_hot_events` now `log::warn!`.
- CLI init in `main.rs`: `logforth::starter_log` with Stderr appender + env filter
  (`RUST_LOG`), `KINORA_TRACE=1` installs `ConsoleReporter`, single root span
  `kinora.main`, `fastrace::flush()` before every return path.
- Out of scope (deferred): rootcause migration, a `--trace` flag, span-assertion
  tests. 301 + 81 tests pass, zero warnings.
