# Plan: Dogfood — extend stencil api-kinograph to the full crate

**Task:** [[kinora-3guj:task]]

Resolve kinograph granularity first, then dogfood module-by-module, validating
the riskier patterns (fn signature/body split, backtick-bearing docs) early.

## Phase 0 — Resolve kinograph granularity *(done)*
Decide per-module vs one crate-wide stencil-api kinograph.

> **Done.** Per-module/per-file binding chosen: the in-place file-binding model
> means a single crate-wide kinograph bound by one file would make every other
> file report all foreign items as unslotted. One kinograph per module.

Dogfood spec.rs (struct, enum, and 4 methods) to first exercise the fn
signature/body split.

> **Done.** `stencil-spec-api` (6 items); sync 6 created; compiles; 72 tests;

> clean. Surfaced + handled the 4-backtick fence case (a definition whose doc
> contains ` ```rust `).
