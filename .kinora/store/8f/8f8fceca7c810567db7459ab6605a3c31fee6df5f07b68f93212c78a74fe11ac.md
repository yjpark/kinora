# Task: `kinora compact` command (single flat root)

**Goal.** Implement manual compaction — read all hot events, derive a single flat root kinograph version deterministically, store it, and update the `roots/<name>` pointer file.

**Realizes:**
- kino://112c90486eba00f02a3fb6cc09c9b7023ca029756bb6a3c7c93920f0e316f020 — Compaction is a manual, deterministic, main-only operation.
- kino://7d1a10e95dcee077fe2cc80f492c9f6ff299b490342800ab65f80f811a35e894 — Canonical history is a chain of root kinograph versions.

**Done when:** genesis (no prior root) produces empty `parents[]`; subsequent compaction links `parents = [prior]` with a new version hash; a no-event compact is a no-op (`new_version = None`, pointer unchanged); two independent compactions over the same hot-event set produce byte-identical blobs; and `kinora compact` prints the expected line and exits 0.

## Outcome
Resolved. Added `kinora::compact` (`compact()`, `build_root()`, `read_root_pointer()`, `CompactParams`/`CompactResult`/`CompactError`) plus `paths::roots_dir`/`root_pointer_path`, and the `kinora compact [--root NAME]` CLI with git-resolved author + RFC3339 ts. Root-kind events excluded from the root's own entries (history chains via event `parents`). Pointer file is exactly the 64-hex hash (read path tolerant of trailing newline). Forks error as `MultipleHeads`; zero-head cycle is a dedicated `NoHead`; root-name path-traversal guarded. 14 new tests; 228 lib + 45 CLI green. Commits `8e1be7e`, `c9d4e19`, `50a5459`.
