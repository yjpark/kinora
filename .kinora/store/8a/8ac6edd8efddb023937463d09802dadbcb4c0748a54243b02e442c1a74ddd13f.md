# Task: cross-root integrity (external refs prevent GC drops)

**Goal.** Enforce that a kino referenced via composition from a different root cannot be GC'd by its owning root — making cross-root composition safe even when the owning root has an aggressive policy.

**Realizes:**
- [[kinora-xi21:spec-5]] — Root ownership is exclusive and structural; kinos move via assign events.

**Plan:** [[kinora-hxmw:plan]]

**Done when:** during `compact_root(B)`, `(id, version)` pointers referenced by any other root's composition kinographs are treated as implicit pins (surviving GC and hot-prune); `compact_all` computes the external-ref snapshot once per batch; only composition (`kind="kinograph"`) references count; circular references don't loop; removing the reference re-enables GC; and the CLI surfaces the retention on the compact line.
