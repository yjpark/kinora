# Hot ledger is one immutable file per event

Settles how kinora records unpromoted history. Replaces the per-lineage append file (`.kinora/ledger/<lineage>.jsonl`) with a content-addressed, branch-safe mempool.

## The decision
- **Chosen: one file per event, sharded by event hash (`hot/<ab>/<event-hash>.jsonl`), over a per-lineage append-only JSONL file.** Each store call writes a single immutable file. Merging two branches is a set-union of files — zero JSONL conflicts, no branch-name coupling. Readers union all hot events and dedup by event hash. The event hash is BLAKE3 of the canonical JSON line; identical logical events dedup across branches.
- **Rejected: per-lineage `ledger/<lineage>.jsonl` append files.** They conflict on merge, couple history to branch names, and require HEAD bookkeeping. HEAD becomes meaningless once kinos are discovered by scanning hot events.
- **Not a dead end:** legacy `ledger/` files can be read at load time during transition; a later phase converts or drops them. The reader stayed dual-layout until the legacy path was retired.

## What this built
- The `hot/<ab>/<event-hash>.jsonl` layout, canonical event encoding + `Event::event_hash()`, crash-atomic `write_event` (tmp+rename) and `read_all_events` (shard-scan + dedup). Verified by write/read roundtrip, dedup idempotence, cross-branch merge simulation, and legacy-coexistence tests. See kino://72f72fd03c48c06bc512ee333431c5a2a6aa12843ef75c9e09bb3d20a89e68da.
- The legacy ledger code was later removed wholesale once nothing wrote to it. See kino://5a7246cbc618b8672bb7dedcc4a83eef513cdc6c06a38f5a67376bda1f67e4ce.

## Consequences / follow-ups
- Prerequisite for every later xi21 phase. Independent of the root model — stands alone as a merge-safety improvement.
- HEAD retired as a write target; forks always surface as `MultipleHeads` rather than HEAD-based tiebreaks.
- The hot ledger directory was later renamed to `staged/`; see kino://112c90486eba00f02a3fb6cc09c9b7023ca029756bb6a3c7c93920f0e316f020 (main-only compaction) for the commit discipline that drains it.
